# Website Brief: thefreeweb.tvl.tech

## Site type
Landing page

## Purpose
Explain and promote The Free Web — a free, open-source system by TVL Technology that lets anyone build a website using Claude AI and GitHub Pages.

## Primary CTA
"Get Started Free" → links to the GitHub repo (https://github.com/tvl-tech/free-website)

## Secondary CTA
"See the Docs" → links to QUICKSTART.md or GitHub README

## Sections
1. **Hero** — bold dark background, headline, subheadline, two CTAs, "takes 15 minutes" hook
2. **How it works** — 5-step process with numbered steps and icons
3. **Features** — 3–4 cards: AI Researcher, AI Builder, Free Hosting, Custom Domains
4. **What it builds** — grid of site types: landing page, portfolio, personal, blog, company
5. **Built by TVL** — short TVL Technology credibility section with company info
6. **Footer** — TVL logo, links, copyright

## Design
- Dark hero section (navy #0f172a)
- White/light body sections
- Accent: indigo #6366f1 for CTAs and highlights
- Secondary accent: cyan #06b6d4 for step numbers and icons
- Font: Inter (Google Fonts)
- Professional, modern, minimal — matches TVL.tech aesthetic

## Key messages
- "No coding. No hosting costs. No credit card."
- "Claude AI researches your background, writes your copy, and publishes your site."
- "Built on GitHub Pages — trusted by millions of developers."
- "An open-source project by TVL Technology."

## Do NOT include
- Contact form (not needed for this landing page)
- Pricing section (it's free)
- Blog
